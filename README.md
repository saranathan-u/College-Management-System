# College-Management-System

Hello Everyone!!!

I developed a full-stack web application using Java, Springboot, Hibernate, and MySQL for managing student, faculty, and course information efficiently.

Designed an intuitive and response interface with HTML, CSS, and JS, ensuring smooth navigation and accessibiltity across devices.

Implemented role-based authentication and CRUD operations, ensuring secure and structured access for Admin, Faculty, and Students.


FEATURES

Enrollments:
1. Courses Management
2. Subjects Management
3. Students Management
4. Faculties Management


INSTALLATION

1. Import this project into your IDE
2. Create new Database and set name as 'collegedate' after that import 'collegedb.sql' file(Attached with this folder) in this database.

Database Connection

* I have used following data for database connection

> url="jdbc:mysql://localhost:3306/Collegedb";
> user name="root";
> password="";

* if you want to change this data then open
".\src\collegeapplication\common\DataBaseConnection.java"
